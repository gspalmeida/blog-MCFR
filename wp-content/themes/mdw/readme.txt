== Description ==

Aesthetics and simplicity are the main features of Material Design - design language created by Google

WordPress has created an online platform to support you in creating free website or a blog.

We present you framework combining the best features of both of them - Material Design for WordPress.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Click on "Upload Theme" button.
3. Download MDW from https://mdwp.io and download the newest package.
4. Choose zip file and click "Install Now" button
4. Go to https://mdwp.io for a guide to customize this theme.
5. Navigate to Appearance > Customize in your admin panel.
